# 📊 Indicateurs PineScripts

## 🧠 Description

Ce repository contient mes scripts PineScript utilisés sur TradingView pour analyser les marchés financiers.

Je m'intéresse particulièrement à la création d'indicateurs techniques simples, efficaces et compréhensibles.

## ⚙️ Contenu

* Indicateurs basés sur RSI
* Logiques de tendance
* Expérimentations personnelles en trading algorithmique

## 🚀 Utilisation

1. Ouvrir TradingView
2. Aller dans Pine Editor
3. Copier-coller le script
4. Lancer l’indicateur sur un graphique
